# SubtitleKit

Package for creating, modifying, and managing subtitle files, such as SubRip (.srt).

## Supported formats

| Format | File extension | Is supported |
| --- | --- | --- |
| SubRip | .srt | Yes |
| WebVTT | .vtt | No |
| SubViewer | .sub | No |
| YouTube Captions | .sbv | No |
| TTML | .ttml | No |
| DXFP | .dxfp | No |
